## Jak urzywać zaklęć

To jest przykładowa konfiguracja do ItemsAdder.

{% hint style="warning" %}
To wymaga **ItemsAdder** 2.5.2+
{% endhint %}

```yaml
  ruby_pickaxe:
    display_name: display-name-ruby_pickaxe
    permission: ruby_pickaxe
    resource:
      material: DIAMOND_PICKAXE
      generate: true
      textures:
      - item/ruby_pickaxe.png
    enchants:
    - ambit:7
```
